# Data download instructions

The raw data used in this analysis are public Fermi data products.

For each GRB, download the Fermi-LAT photon event file and spacecraft file from the Fermi Science Support Center LAT Data Server.

Use one folder per GRB:

- GRB080916C
- GRB090510
- GRB090902B
- GRB090926A
- GRB130427A

Each folder should contain one photon file ending in `_PH00.fits` and one spacecraft file ending in `_SC00.fits`.

The GRB trigger times, positions, redshifts, and photon-selection criteria are listed in the notebook and manuscript.
